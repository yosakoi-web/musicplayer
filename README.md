# 練習用音楽プレーヤー

GitHub Pages向けの練習用音楽プレーヤーです。曲ボタンを押すと、指定されたMP3をすぐに再生します。

## 音源の入れ方

`music` フォルダへ、次のファイル名のまま12曲を入れてください。

1. `01 祭花.mp3`
2. `02 時代(とき)に燃えて恋しぐれ.mp3`
3. `03 toki make love it 2023.mp3`
4. `04 土岐旋風歌.mp3`
5. `05 春駒わっしょい.mp3`
6. `06 南中ソーラン.mp3`
7. `07 GIFU 今ここで.mp3`
8. `08 よっちょれ.mp3`
9. `09 ミナモダンス.mp3`
10. `10 ダンシングヒーロー.mp3`
11. `11 うらじゃ音頭.mp3`
12. `12 世界に一つだけの花.mp3`

大文字、小文字、空白、括弧を含め、上記と同じファイル名にしてください。

## GitHub Pagesで公開する方法

1. このフォルダ内のファイルと、音源を入れた `music` フォルダをGitHubリポジトリへアップロードします。
2. リポジトリの `Settings` を開きます。
3. `Pages` を開きます。
4. `Deploy from a branch` を選択します。
5. Branchを `main`、フォルダを `/ (root)` にして保存します。
